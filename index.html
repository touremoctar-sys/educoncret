🎫 Le Passeport 48H

Maîtrisez une compétence professionnelle en 2 heures

La méthode narrative pour apprendre l'essentiel, tout de suite.

Le problème

Vous avez besoin d'acquérir une compétence précise, mais :

⏰

Les formations longues : 2 à 5 jours, trop cher, trop long, pas dispo quand vous en avez besoin.

📺

Les tutos YouTube : Vous passez 3h à chercher l'info utile dans 10 vidéos contradictoires.

📖

Les livres blancs : 50 pages de blabla pour 2 pages utiles.

La solution : L'Immersion Narrative de 2 Heures

J'ai créé Le Passeport 48H, un format de formation express qui vous donne l'essentiel actionnable sur une compétence précise.

  • ✅ Vous choisissez votre destination parmi un catalogue de compétences.
  • ✅ Vous vivez une session individuelle de 2 heures (visio).
  • ✅ Vous repartez avec une "Carte d'Action" : un document A4 qui résume tout.

La signature pédagogique

On ne vous dit pas : "Voici les 7 principes du copywriting."

On vous dit :
"Vous êtes chargé d'écrire la page d'accueil. Le visiteur arrive. Qu'est-ce qu'il doit ressentir dans les 3 premières secondes ? Voici les 3 leviers narratifs pour le capturer."
