Positionnement

Je conçois et déploie des dispositifs d’apprentissage centrés sur l’individu, visant le développement des compétences, la structuration de projets professionnels et l’autonomisation des apprenants. Mon approche repose sur une ingénierie pédagogique rigoureuse, combinée à un accompagnement individualisé, permettant de transformer une intention en projet concret et opérationnel.

🎯 Missions principales

  • Analyse des besoins des apprenants et des porteurs de projet
  • Conception de parcours pédagogiques personnalisés
  • Structuration de dispositifs d’accompagnement à visée professionnelle
  • Facilitation de l’émergence et du développement de projets
  • Suivi individualisé et évaluation des compétences
  • Accompagnement socio-éducatif vers l’autonomie et la prise de décision

📲 Version application

Le site peut fonctionner comme une application web

J’ai ajouté les lignes nécessaires pour préparer une expérience installable sur mobile et desktop, avec une base PWA propre.

Manifest

Le site déclare un fichier manifest.webmanifest pour définir le nom, les icônes, la couleur du thème et le mode d’affichage.

Installation

Un bouton d’installation apparaît si le navigateur supporte l’installation de l’application web.

Icône d’écran d’accueil

Le site est prêt pour iPhone et Android avec une icône d’application et des métadonnées mobiles.

Service Worker

Le script enregistre un service-worker.js pour préparer le cache et améliorer l’expérience hors ligne.

// À placer à la racine du site : /manifest.webmanifest
{
  "name": "EduConcret — Moctar Touré",
  "short_name": "EduConcret",
  "start_url": "./",
  "display": "standalone",
  "background_color": "#f7f3ed",
  "theme_color": "#0e0c0a",
  "icons": [
    { "src": "icons/icon-192.png", "sizes": "192x192", "type": "image/png" },
    { "src": "icons/icon-512.png", "sizes": "512x512", "type": "image/png" }
  ]
}

// À placer à la racine du site : /service-worker.js
self.addEventListener('install', (event) => {
  event.waitUntil(caches.open('educoncret-v1').then(cache => cache.addAll(['./'])));
});
self.addEventListener('fetch', (event) => {
  event.respondWith(caches.match(event.request).then(response => response || fetch(event.request)));
});
